Welcome to the Security Demo Website

This is a demonstration of how sensitive information can be leaked by exposing credentials in the page source code.

What You Should Do:

To see the exposed credentials, simply **view the page source**.

In most browsers, right-click on the page and select **"View Page Source"** (or press **Ctrl+U**). The credentials will be visible there.

WARNING:

This is a **fake** demonstration page. Do not expose real credentials like this!
